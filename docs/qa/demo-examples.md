# Agent Exchange Demo Examples

Short sample legal tasks for testing the AEX Demo UI. These are concise versions designed for quick demos.

---

## Sample 1: NDA Review (5 pages)

**Copy this text:**

```
Review this NDA: Term is 3 years with 5-year survival of confidentiality obligations. Receiving Party must protect info with "reasonable care" and return/destroy all materials on termination. Disclosing Party can seek injunctive relief without posting bond. Delaware law governs.

Identify: (1) terms favoring Disclosing Party, (2) liability exposure, (3) overly broad definitions, (4) recommended modifications.
```

---

## Sample 2: Employment Contract (8 pages)

**Copy this text:**

```
Review this employment contract: 24-month non-compete across North America. Full IP assignment for all work during employment. At-will with 2-week severance only. Mandatory arbitration in SF, waiving jury trial. Employee must give 30-day notice; employer can terminate immediately.

Analyze: (1) non-compete scope, (2) IP assignment concerns, (3) severance adequacy, (4) arbitration implications, (5) negotiation points.
```

---

## Sample 3: SaaS Agreement (3 pages)

**Copy this text:**

```
Review this SaaS agreement: 36-month initial term, auto-renews unless 90-day notice given. Fees increase 10% each renewal. Provider liability capped at 12 months fees. 99.5% uptime SLA with credits capped at 30% annually. Customer data deleted 30 days after termination.

Flag: (1) auto-renewal traps, (2) data privacy gaps, (3) SLA adequacy, (4) liability limits, (5) termination issues.
```

---

## Sample 4: Vendor Contract (6 pages)

**Copy this text:**

```
Review this vendor contract: 24-month term, $25K/month. Deliverables are work-for-hire owned by client. Vendor liability capped at 6 months fees. GDPR/CCPA compliance required. 24-hour breach notification. Client can terminate for convenience with 90 days notice plus 3-month fee.

Evaluate: (1) scope creep risks, (2) IP ownership, (3) liability caps, (4) termination costs, (5) data protection.
```

## Quick Test (Recommended for Demo)

**Copy this text:**

```
Review this software license: Perpetual license for $50K. No source code access. Vendor provides "reasonable efforts" support. Liability limited to license fees paid. No warranties except title. Customer cannot assign without consent.

Identify risks and recommend modifications.
```
